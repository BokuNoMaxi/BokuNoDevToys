# BokuNoDevToys

SvelteKit-Projekt mit Dev-Tools im Browser.

## Nach jeder Änderung

Nach jeder Codeänderung NPM neu starten, damit die Änderungen sichtbar werden:

```bash
npm run dev
```

Der Dev-Server läuft auf Port 5173. Falls er bereits läuft, muss er neu gestartet werden:

```bash
pkill -f "vite" && npm run dev
```
